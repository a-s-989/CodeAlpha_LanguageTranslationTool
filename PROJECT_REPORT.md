## Project Overview

The Language Translation Tool is a desktop application developed using Python and Tkinter GUI framework. 
It allows users to translate text from one language to another using the Google Translator API (via the deep_translator package). 
Additional functionalities include copying the translated text to the clipboard and converting the translated text to speech.

---

## Features

- Translate text between multiple languages
- Copy translated text to clipboard
- Speak the translated text using Text-to-Speech
- Simple and easy-to-use GUI

## Installation and Usage

### Installation

1. Clone the repository or download the ZIP file and extract it.
    ```bash
    git clone https://github.com/your-username/Language-Translation-Tool.git
    cd Language-Translation-Tool
    ```

2. Install required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

### Running the Project

Run the Python script using:
```bash
python translator_app.py
```
